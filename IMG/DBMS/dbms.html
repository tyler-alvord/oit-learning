This document is for the Maria DB image. It will describe the basic features of the dbms qcow as well as some of it's maintenance and use cases.

DESCRIPTION:

This image is for use as a connection point for students that want to work with an external database with their systems. It will be useful for students that want to query live data, or for students that want to create a database with non-persistant data for testing.

INSTALLATION PROCEDURE:

  1. Start with the clean installation of lubuntu
  2. Install Mariadb with the following command
    • sudo apt-get install mariadb-server
  3. Run through the automated configuration for MariaDB with the following command:
    • sudo mysql_secure_installation
    • Accept the default configuraiton presented by each prompt
  4. Download the database script from github
  5. Navigate to the download directory where the database script is stored and run the script using the following command:
    • sudo mysql < employees.sql
  6. Run the following commands to validate that the server is up and running without a problem
    • sudo mysql -u root -p
    • USE employees
    • SHOW TABLES

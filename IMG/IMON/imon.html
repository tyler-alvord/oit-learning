Admin Info: Infection Monkey Image

DOCUMENT INFORMATION:

This document is for the Infection Monkey (imon) image. It will describe the basic features of Infection Monkey, as well as some of the image's maintenance and use cases.

DESCRIPTION:

Infection Monkey is a vulnerability scanner and network attack simulator. It uses one machine (in this case, this image - also known within the software as "monkey island") to propogate access to the network through exploits and attack methods.

This image should be used after consent to the rules of engagement - using this image for unintended purposes is dangerous, traceable, and in violation of not only the terms of use for this system, but also the terms of service for the application.

Accessing Infection Monkey

Infection monkey will be located at https://localhost:5000/

INSTALLATION PROCEDURE:

  1. Create a guardicore account at https://www.guardicore.com/infectionmonkey/
  2. Download the debian installer from the email that they will send you on the lubuntu machine (base image)
  3. Follow the Infection Monkey install guide
  4. Set the admin account to the OIT admin credentials
  5. Head to the Infection Monkey Admin interface and verify that the image is up and running.

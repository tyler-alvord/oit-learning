Welcome to the ADMIN folder - this folder contains information about the maintenance and administration of the OIT learning envrionment. 
The IMG folder is used to describe the configurations for all of the images that are stored in the system. All necessary steps to recreate the images should be contained there. 
The META folder is used to describe maintenance and administration of the two ubuntu servers that host the Openstack environment. This includes the base configurations for the operating system, installed packages, admin contact list, and policy descriptions and rules. 
The TMPL folder is used to store templates for images and documentation

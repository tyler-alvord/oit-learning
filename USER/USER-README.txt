Welcome to the USER folder! Here you'll find all you need to get started as a user of the OIT Openstack Learning environment!

Use the IMG folder to get familiar with how to use the images once you've instantiated them. Things like login information, how-to's and rules of use with specific images will be found here. 

Use the META folder to build an understanding of Openstack. How to login, how to start up instances, and other tips on operating in the learning environment.

Pro tip: Check out the "Getting Started Guide" in the USER/META folder to get a feel for things!
